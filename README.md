# i3stylepy
i3-style replacement (compatible with its themes) because I don't want to have node.js /coffeescript installed for using i3.

## Usage
```
i3stylepy[master*] % python3 i3stylepy.py --help
usage: i3stylepy.py [-h] {apply,list,extract} ...

positional arguments:
  {apply,list,extract}

optional arguments:
  -h, --help            show this help message and exit
```

### apply
```
i3stylepy[master*] % python3 i3stylepy.py apply -t themes/solarized -c ~/.config/i3/config -o solarizedconf    

```

### extract
```
i3stylepy[master*] % python3 i3stylepy.py extract -c solarizedconf -o solarizedtheme.yaml

```
